# BarcodeBite Privacy Policy

Effective date: March 1, 2026

## Data Collected
- Account email and hashed password
- Product scan history and timestamps
- Subscription status and plan metadata
- Crash diagnostics and usage analytics (if enabled)

## Purpose
- Provide barcode analysis and history features
- Secure authentication and account access
- Maintain subscription entitlements
- Improve stability and product quality

## Sharing
- No sale of personal data
- Data processors may include cloud hosting, analytics, crash reporting, and payment providers

## Retention
- Account data is retained while account is active
- Scan history can be deleted by user action
- Logs and diagnostics retained per operational need

## User Rights
- Access and correction requests
- Account deletion request
- Data export request
